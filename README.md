# Apex Auto Mods

Apex Auto Mods is a full-stack car customization web app (React + Node.js + MongoDB).

Quick overview

- Frontend: interactive 3D car customizer, auth, marketplace, dashboard.
- Backend: JWT auth, services, builds, MongoDB storage.

How to clone & run (dev)

```bash
# Clone repository
git clone https://github.com/RividuPesara/apex-auto
cd apex-auto-mod

# Backend
cd backend
npm install
npm run dev

# Frontend
cd ../frontend
npm install
npm run dev
```
